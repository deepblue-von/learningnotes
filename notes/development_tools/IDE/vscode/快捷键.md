| --------------- | -------------------------------- |
| 操作            | 效果                             |
| Ctrl shift E    | 切换到文件栏                     |
| Ctrl B          | 隐藏边栏                         |
| Ctrl shift `    | 打开终端                         |
| ctrl p          | 快速查找文件                     |
| ctrl shift p    | 打开设置命令行                   |
| ctrl shift K    | 删除行                           |
| ctrl tab        | 在打开的文件之间进行切换         |
| ctrl `          | 打开下方面板                     |
| ctrl /          | 注释一行                         |
| alt shift A     | 块注释                           |
| shift + tab     | 整体左移                         |
| shift + alt + ⬇ | 向下复制一行                     |
| function.var    | 函数有返回值时，可以直接test.var |
| ctrl alt N      | 运行程序                         |
| ctrl alt M      | 结束进程                         |
