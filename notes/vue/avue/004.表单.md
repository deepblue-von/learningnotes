### 表单

#### 自定义表单	需要注意的事项

**表单自定义列**：

```js
{
    label: '脸部信息',
    prop: 'face', 
    type: 'upload',
    listType: 'picture-img',
    formslot: true,  // 自定义列需要在slot前面加上form
    disabled: true
    }]
}
```

```vue
<!--如果在avue-form中插槽中写slot即可-->
<avue-form :option="option" v-model="reviewDetail" @submit="handleSubmit">
    <template slot-scope="scope" slot="face">
     <el-image :src="reviewDetail.face" style="width:180px; height:180px" alt=""                         :fit="fill" :preview-src-list="reviewDetail.face"></el-image>
    </template>
</avue-form>
```



```vue
<!--如果在avue-crud中插槽中要写slotForm-->
<avue-crud :option="option" v-model="reviewDetail" @submit="handleSubmit">
    <template slot-scope="scope" slot="faceForm">
     <el-image :src="reviewDetail.face" style="width:180px; height:180px" alt=""                         :fit="fill" :preview-src-list="reviewDetail.face"></el-image>
    </template>
</avue-crud>
```
