# ps快捷键

| 快捷键                 |                    |
| ---------------------- | ------------------ |
| Ctrl + J               | 复制图层           |
| Ctrl + T               | 变换控件           |
| Alt + delete           | 填充颜色           |
| Ctrl                   | 自动选择           |
| Ctrl + D               | 取消选区           |
| Alt                    | 圆心画圆           |
| space                  | 移动画布           |
| ctrl + n               | 新建文件           |
| ctrl + w               | 关闭文件           |
| ctrl + k               | 首选项             |
| alt + 新建图层         |                    |
| alt + 显示图层（眼睛） | 这显示改图层       |
| alt 拖动图层           | 复制图层           |
| 空格键不松             | 抓手工具，移动视图 |
|                        |                    |
