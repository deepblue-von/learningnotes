## 九九乘法表

```python
# 正常打印乘法表
for i in range(1, 10):
    s = ""
    for j in range(1, 10):
        if j <= i:
            s += str.format("{0:1}*{1:1}={2:<3}", i, j, i*j)
    print(s)
    
# 倒三角打印乘法表
for i in range(1, 10):
    s = ""
    for j in range(1, 10):
        if i <= j:
            s += str.format("{0:1}*{1:1}={2:<3}", i, j, i * j)
        else:
            print("       ", end="")
    print(s)
```
