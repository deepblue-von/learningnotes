### js函数定义

**函数声明**

```js
function functionName(parameters) {
  执行的代码
}
```

**函数表达式**

```
var x = function (a, b) {return a * b};
```

### js对象

```js

person={firstname:"John",lastname:"Doe",age:50,eyecolor:"blue"};
```

#### for...in 循环遍历person对象的每一个属性

```js
for (x in person)
{
    txt=txt + person[x];
}
```
